## Convex Loading Flicker Fix

- Time: 15:45 CEST, approximate duration 15 minutes.
- User reported the new APK shows `Clubs werden geladen...` and briefly flickers to `Keine Clubs gefunden`, but never finishes loading data.
- Root cause: `usePublicConvexQuery` used the generated Convex `api.*` function reference directly as a React effect dependency. These references come from a Convex proxy and can be unstable across renders, causing the HTTP query to restart repeatedly.
- Fixed `src/hooks/use-public-convex-query.ts` to use Convex's stable function name as the query key, recreate a stable function reference with `makeFunctionReference`, keep stable args by serialized args key, and ignore stale overlapping responses.
