- Time: 12:06 CEST, approximate duration 15 minutes.

- Reviewed prior session summaries before choosing a build path. Relevant prior context:

- Updated `README.md` with a concise pointer to the phone APK workflow.
- Decision: use a GitHub-hosted Gradle APK workflow first because it avoids requiring Expo/EAS account setup for simple internal Android testing. EAS remains a later option for managed signing, distribution, or store-oriented builds.
